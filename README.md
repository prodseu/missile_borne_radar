# missile_borne_radar

Notes and references on missile-borne radar advantage regions and penetration jamming.

This repository is a research-note project rather than a software package. It organizes formulas, figures, and written notes related to radar operating bands, detection range, jamming effects, and penetration scenarios.

## Reading

Start with [`missile_borne_radar.md`](missile_borne_radar.md). The document is written in Markdown and uses LaTeX-style formulas and local image references.

For the best reading experience, open it in a Markdown editor that supports:

- table of contents rendering;
- LaTeX math preview;
- local image preview.

## Topics

- Radar detection range and radar-equation-based reasoning.
- Frequency-band effects on seeker/radar performance.
- Active jamming and stand-off / escort jamming scenarios.
- Penetration and anti-jamming concepts.
